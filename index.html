Welcome to the Media Server API

This API allows you to view, add, update, and delete items in three categories:

  • /movies — List of movies (title, year)
  • /series — List of series (title, seasons)
  • /songs — List of songs (title, artist)

Supported Methods:

  • GET — View all items
  • POST — Add a new item (JSON body)
  • PUT — Update an item (JSON body with existing title)
  • DELETE — Remove an item (JSON body with title)

Example JSON (POST/PUT):

{
  "title": "The Matrix",
  "year": 1999
}

To delete:

{
  "title": "The Matrix"
}
